Steps ... !!!

  1. Create folder and create index.html
  2. `npm init` - init npm
  3. package install - `npm i lite-server`
  4. lite server configure - copy paste command in script tag in package.json `"dev": "lite-server"`
  5. run the project using `npm run dev` command
  6. install bootstrap `npm i bootstrap@5.3.0-alpha3`
  7. configure bootstrap add link and script tag and pass node_module
